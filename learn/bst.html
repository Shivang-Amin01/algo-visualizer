Learn: Binary Search Tree (BST)

A Binary Search Tree (BST) is a binary tree where each node has:

  • Left subtree with values less than the node’s value
  • Right subtree with values greater than the node’s value

BST Operations

Insertion

if (value < node.value): go left
else: go right

Search

Same logic as insertion — go left or right based on comparison.

Deletion Cases

  • No child → just delete
  • One child → link parent to child
  • Two children → replace with inorder successor

Time Complexity

  • Average: O(log n)
  • Worst-case (unbalanced): O(n)
